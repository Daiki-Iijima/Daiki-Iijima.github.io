テキストレイアウトについて

参考URL

w3schools.com

strong(太文字)

  • <b></b>で表現をすることもできるが、HTML5からは、<strong></strong>タグが推奨されている
    • 人が目視で確認する時に、<strong></strong>の方が読みやすいから

em(イタリック)タグ

  • <i></i>で表現することもできるが、HTML5からは、<em></em>タグが推奨されている
    • em = Enphasize

ins(下線)タグ

  • <u></u>で表現でこともできるが、HTML5からは、<ins></ins>タグが推奨されている
    • ins = insert

sup(上付き文字)タグ

  • sup = Superscript

    sub(下付き文字)タグ

  • sub = Subscript

    del(打ち消し)タグ

  • del = delete

    hr(区切り線)タグ

  • 自己完結型(</hr>がいらない)
  • 勝手に改行が入る
  • hr = Horizontal Rule

    mark(ハイライト)タグ

  • mark = marking
