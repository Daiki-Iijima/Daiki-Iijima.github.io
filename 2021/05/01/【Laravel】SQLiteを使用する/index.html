設定項目の解説

connectionsという配列の中のsqlite配列にsqliteの設定があります。

  • 'driver' => 'sqlite' : ドライバー名
  • 'url' => 'env('DATABASE_URL')' : DATABASE_URL環境変数を取得する
  • 'database' => env('DB_DATABASE', database_path('database.sqlite')) : データベースの値を変更する
  • 'foreign_key_constraints' => env('DB_FOREIGN_KEYS', true) : DB_FOREIGN_KEYS環境変数が取得して、ない場合、trueを返す
