## git commit 時に-aスイッチを使う
git addをするのが面倒くさいときは、commitと同時にステージンをするためのコマンドが用意されている
	- `-a`スイッチで、自動ステージングされるのは、すでにGitが認識している(新規追加ではない)ファイルのみを自動でステージングする

```shell-session
$ git commit -a -m "自動ステージングコミット"
```
