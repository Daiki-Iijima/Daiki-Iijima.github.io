# 改行の入れ方3パターン
## 1. 半角空白を２つ入れて改行する
- この改行の入れ方の場合、ブロックは別れない
- `HTML`で言うところの、`<br>`タグと同じ
```markdown
1
2
3
```

## 2. 改行を１つ入れる
- ブロックが別れる
- `HTML`で言うところの、`<p>`タグと同じ
```markdown
1

2

3
```

## 3. \<br\>タグを入れる
- `HTML`のタグと同じ
```markdown
1<br>2<br>3<br>
```
