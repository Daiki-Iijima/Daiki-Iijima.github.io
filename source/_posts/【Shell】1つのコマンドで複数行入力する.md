#目次

# echoコマンド
通常`echo`コマンドを使用すると、1行をファイルに追記できます。


新規ファイルに文字列を記述する場合
```bash
$ echo "テスト" > t.txt
```

既存のファイルに追記する
```bash
$ echo "テスト" >> t.txt
```

# 複数行をいっぺんに入力する「ヒアドキュメント」
ヒアドキュメントは特定の文字列が入力されるまで、まとめて入力としして扱う機能です。

```bash
$ cat << '終了文字列' > ファイル名
```

## 使用例
t.txtに`EOF`が入力されるまで入力された文字列を追記する

```bash
$ cat << 'EOF' > t.txt
```
