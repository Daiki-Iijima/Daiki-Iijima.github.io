# コミットメッセージから探す
`--grep`スイッチを使用して`=`に指定した文字列と一致するコミットメッセージのコミットを表示することができます。

```bash
$ git log --grep=test
```

# コミットした人物から探す
`shortlog`でそのリポジトリにコミットした人ごとのコミットを表示して、その後`--ather`スイッチで人物を指定して指定した人物のコミットだけを表示させる

- `shortlog`でメールアドレスを表示させるには`-e`スイッチをつける

## 実行例
ユーザーごとにコミットが表示される
```bash
$ git shortlog

Daiki Iijima(10)
	....
	....
	....

Kojima Yui(21)
	...
	...
	...
```

個人を指定して表示させる
```bash
$ git log --author="Daiki Iijima"
```
