# 判定方法
```php
if(is_null($getValue) || $getValue === ''){
  //  未入力
}else{
  //  入力されている
}
```

## 解説
- `is_null`関数は、`変数が未定義 or 値がnull`のときに`true`を返します。
- `=== ''`は、`厳格な比較`を行い変数が空の場合に`true`を返します。
